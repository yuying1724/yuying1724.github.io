【心得】Github 免費架站術！輕鬆打造個人品牌

By ying 06 February 2023

契機

曾經喜歡透過部落格來紀錄和分享生活中大小事， 但經由 無名小站、痞客邦、Google Blogger 來紀錄生活與學習筆記， 但因為時代的演變，很多的某些平台不見了！

然而剛好因為想要加強自己的能力，購買了Java 工程師必備！Spring Boot 零基礎入門 的課程。 然後看到老師也有開 Github 免費架站術！輕鬆打造個人品牌 課程。

我被標題吸引了！！！

身為一個工程師，當然要用工程師的方式來展示自己啦！

我就是這麼膚淺！ 身為工程師的傲嬌，的確有想法想要自己寫部落格，但… 其實我的能力真的很普通，還是希望有人能帶領我怎麼做， 就這麼膚淺的原因購買了此課程。

學習歷程

其實買了之後，我的惰性依舊。過了很久才開啟這門課， 但不知道是不是我心中本來就想做一個部落格的想法在萌芽！ 跟著課程看了之後信心大增，照著老師的步驟沒想到真的順利弄出了雛型。 課程看到Hugo的文章是使用Markdown來撰寫，和我的程式筆記hackMD語法相同， 更讓我想趕快完成部落格，我想把我的筆記記錄在部落格，也能透過部落格分享給有需要的人。

不過也有點受挫，我看不太懂hugo的程式語言，前端的知識也不夠充足， 其實有很多東西可以試著去調整，真心覺得我懂的東西太少了，

心得

真心推薦如果想要自己挑戰架設一個部落格可以參考這門課， 跟著步驟做下去一定可以順利完成的！ 而跟著做完我個人是非常的有成就感，尤其是整個部落格都可以依照自己的想法， 不會被平台限制，透過自己的創意靈活彈性製作專屬的網站。

不過如果只是單純想寫文章，我建議直接找一般的部落格平台就夠了。 畢竟這個我覺得還是要懂一點點程式語言或許會比較好上手一點， 而我自己其實也只會點皮毛，所以還是花了點時間熟悉和理解。

也期許自己能夠透過這個部落格來記錄我的筆記以及生活。

成果

這個部落格就是我跟著這堂課操作的作業成果 Ying's Blog
